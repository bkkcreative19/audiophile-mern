## Getting Started




### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bkkcreative19/audiophile-mern.git
   ```
2. Install NPM packages
   ```sh
   npm install in both client and server
   ```
3. Enter your port and mongoDBLink in `.env file`
